# Money Manager Frontend

React-based frontend for Money Manager application.

## Installation

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```
VITE_API_URL=http://localhost:5000/api
```

## Running the Application

Development:
```bash
npm run dev
```

Build:
```bash
npm run build
```

Preview:
```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- React Router
- Tailwind CSS v4
- Chart.js with react-chartjs-2
- Axios
- React Hot Toast
- Lucide React Icons
